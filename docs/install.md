<a name="dependencies">
## Dependencies
[NPM]("https://www.npmjs.com") & [BOWER]("https://www.bower.io")
---
```bash
cd path/to/project
npm install && bower install
```
_Note: Macs/Linux may need `sudo` in front of `npm install`_
